## Locale

Run `ng extract-i18n` once to extract the source language file

Run `ng serve --configuration=fr` to serve the localized version of the app

Run `ng build` to build all localized versions of the app (`dist/i18n/en-US`, `dist/i18n/fr`, etc)

Run `ng build --configuration=production,fr` to build the localized version of the app for production
